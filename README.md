Situation Room
===============

Situation Room which help you to display multiple webpage on a monitor and
control display from IRC. It's usefull for IT team in order to create tactical display
pilot by the team.

Installation
-------------

* Install node.js
* Install npm

* Use install.sh for install dependencies.

Configure
---------

Modify config.js
